INSTRUCCIONES
-------------
Coloca aquí los archivos JPG o WebP con los nombres exactos indicados
abajo. La web detecta automáticamente si la imagen existe:
  · Si existe → la muestra como fondo de la tarjeta.
  · Si no existe → usa el gradiente CSS de color por defecto.
No hace falta tocar ningún código.

FORMATO RECOMENDADO
-------------------
  · Tipo:       JPG o WebP (preferible WebP para menor tamaño)
  · Ancho:      1 600 px (mínimo 1 200 px)
  · Alto:       900 px aprox. (ratio 16:9 o 4:3 funcionan bien)
  · Calidad:    80–85 % (buen equilibrio calidad/peso)
  · Peso máx.:  300 KB por imagen (usa squoosh.app o tinypng.com)
  · Orientación: horizontal (landscape)
